# AI Log Analyzer

This Python project analyzes authentication logs and flags suspicious activity using heuristic AI-style detection.

## Features
- Brute-force detection (failed login threshold)
- After-hours login detection
- New IP login detection
- Impossible travel detection
- Easy to extend for Windows/Linux logs

## Example Usage

1. Install Python 3.11+  
2. Place your log file (`auth.log`) in the repository folder  
3. Run:

```bash
python log_analyzer.py
